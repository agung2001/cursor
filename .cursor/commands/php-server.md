Run a PHP built-in development server on all interfaces at port 8000 using the following terminal command:

```
php -S 0.0.0.0:8000
```

Execute this command in the terminal from the project root directory. The server will be accessible at:
- http://localhost:8000 (local machine)
- http://<your-ip>:8000 (other devices on the network)

To stop the server, press `Ctrl+C` in the terminal.
